PART 1: SIGNING UP FOR SELLERAMP (10 MINUTES)
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Go to SellerAmp Website
--------------------------------

1. Open your browser (Chrome)
2. Go to: https://selleramp.com
3. You'll see the homepage


STEP 2: Start Free Trial
-------------------------

1. Look for button that says:
   "START FREE TRIAL" or "TRY FOR FREE" or "GET STARTED"

2. Click it

3. You'll see pricing plans:
   • Solo: £25/month
   • Pro: £50/month
   
4. Choose "Solo" (it's enough for beginners)

5. Click "Start Free Trial" on Solo plan


STEP 3: Create Account
-----------------------

Fill in the form:

1. Email: your.email@gmail.com
2. Password: (create strong password)
3. Name: Your Name
4. Country: United Kingdom

5. Payment Method:
   → Enter credit/debit card
   → You WON'T be charged during trial (7-14 days)
   → You can cancel before trial ends

6. Click "Create Account" or "Sign Up"


STEP 4: Verify Email
---------------------

1. Check your email inbox

2. Look for email from SellerAmp

3. Click verification link in email

4. This confirms your account


STEP 5: Welcome Screen
-----------------------

After verification, you'll see welcome screen:

1. "Welcome to SellerAmp SAS!"

2. Click "Get Started" or "Install Extension"

3. This takes you to Chrome Web Store

PART 2: INSTALLING SELLERAMP CHROME EXTENSION (5 MINUTES)
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Go to Chrome Web Store
-------------------------------

Either:
   A) Click link from SellerAmp welcome screen
   
   OR
   
   B) Manually:
      1. Go to: chrome.google.com/webstore
      2. Search: "SellerAmp SAS"
      3. Click on "SellerAmp SAS" (official extension)


STEP 2: Install Extension
--------------------------

1. You'll see extension page with:
   • SellerAmp SAS icon
   • "Add to Chrome" button
   • Description
   • Reviews

2. Click "Add to Chrome" button

3. Popup appears: "Add SellerAmp SAS?"
   → Click "Add extension"

4. Extension installs (5-10 seconds)

5. You'll see confirmation:
   "SellerAmp SAS has been added to Chrome"


STEP 3: Pin Extension (Important!)
-----------------------------------

1. Look at top-right of Chrome browser

2. Click the puzzle piece icon (Extensions)

3. Find "SellerAmp SAS" in the list

4. Click the pin icon next to it
   → This keeps it visible

5. Now you'll see SellerAmp icon in toolbar


STEP 4: Log Into Extension
---------------------------

1. Click the SellerAmp icon in toolbar

2. Login popup appears

3. Enter:
   • Email: (your SellerAmp email)
   • Password: (your SellerAmp password)

4. Click "Login"

5. You'll see: "Logged in successfully"

PART 6: UNDERSTANDING EACH SELLERAMP FEATURE
═══════════════════════════════════════════════════════════════════════════════

FEATURE 1: IP ALERTS (Most Important!)
---------------------------------------

What it means:
   "IP" = Intellectual Property
   = Trademarks, brands, copyrights

Why it matters:
   Selling trademarked products without permission = Account SUSPENDED

What SellerAmp shows:

✅ "No IP Complaints"
   → SAFE! You can sell this
   → Brand doesn't restrict sellers
   → Go ahead!

❌ "IP Alert" or "Brand Gated"
   → DANGER! Don't sell this!
   → Brand has trademark protection
   → Amazon will suspend your account if you try

⚠️ "No Data"
   → Unknown
   → Be cautious
   → Do extra research


ACTION:
   If ❌ IP Alert → SKIP IMMEDIATELY
   Never risk your account!


FEATURE 2: RESTRICTIONS
-----------------------

What it means:
   Some categories need Amazon approval ("gated")
   Examples: Baby, Beauty, Groceries

Why it matters:
   Can't list product without approval = stuck with inventory

What SellerAmp shows:

✅ "Not Restricted"
   → SAFE! You can list this
   → Category is open to you
   → Go ahead!

❌ "Restricted - Approval Needed"
   → Can't sell without approval
   → Need to apply to Amazon
   → SKIP unless you have approval

⚠️ "Test Buy Recommended"
   → Uncertain
   → Try adding in Seller Central
   → Check manually


ACTION:
   If ❌ Restricted → SKIP
   Unless you have category approval


FEATURE 3: PROFIT CALCULATOR
-----------------------------

This is the CORE feature!

What it calculates:

1. Sell Price
   → Current Buy Box price
   → What you'll sell for

2. Your Cost
   → What YOU enter
   → Your warehouse price

3. Amazon Fee (15%)
   → Amazon takes 15% of sell price
   → Called "referral fee"

4. FBA Fee
   → Amazon packs & ships
   → Varies by size/weight
   → Usually £2.50-5.00

5. Net Profit
   → Sell Price - Cost - Fees
   → This is YOUR money!

6. ROI
   → Net Profit / Cost
   → Shows % return
   → 50% ROI = You double your money (+50%)

7. Margin
   → Net Profit / Sell Price
   → Shows profit as % of sale


EXAMPLE CALCULATION:

Sell Price: £12.99
Cost: £4.50
Amazon Fee: £12.99 × 0.15 = £1.95
FBA Fee: £3.50
Total Fees: £1.95 + £3.50 = £5.45

Net Profit: £12.99 - £4.50 - £5.45 = £3.04
ROI: £3.04 / £4.50 = 67.6%
Margin: £3.04 / £12.99 = 23.4%

PART 8: TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

PROBLEM: SellerAmp overlay doesn't appear
------------------------------------------

SOLUTION:
   1. Refresh the Amazon page (F5)
   2. Check if extension is installed (puzzle icon)
   3. Check if you're logged in (click SellerAmp icon)
   4. Try different Amazon product page
   5. Restart Chrome


PROBLEM: Can't log into SellerAmp
----------------------------------

SOLUTION:
   1. Check email/password are correct
   2. Check email for verification link
   3. Reset password if needed
   4. Contact SellerAmp support


PROBLEM: Shows "Not Connected to Seller Central"
-------------------------------------------------

SOLUTION:
   1. Open Seller Central in another tab
   2. Log in
   3. Refresh Amazon product page
   4. SellerAmp should detect connection


PROBLEM: Cost field won't accept my number
-------------------------------------------

SOLUTION:
   1. Use decimal point (4.50, not 4,50)
   2. Don't include £ symbol
   3. Just type number: 4.50


PROBLEM: "No Data Available"
-----------------------------

SOLUTION:
   1. Product might be new
   2. SellerAmp hasn't indexed it yet
   3. Do manual check
   4. Consider SKIP if uncertain
